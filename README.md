# RertX -- 新一代 AI 聚合平台

RertX 整合自研 AI 模型（RertMiniX、RertMaX、RertMvX）与全球顶尖大模型（暂未接入），提供极致的多模态对话、API 接入、知识库与工作流能力。

## 技术栈

- **框架**: Next.js 16 (App Router)
- **UI**: React 19 + Tailwind CSS
- **语言**: TypeScript
- **包管理器**: pnpm

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 启动生产服务
pnpm start
```

## 项目结构

```
├── app/          # Next.js App Router 页面与布局
├── components/   # React 组件
├── public/       # 静态资源
└── tailwind.config.ts
```
